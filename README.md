# Android Video Downloader

An Android app for downloading videos from websites.

## Features (Planned)

- 📥 Download videos from various websites
- 🎬 Multiple quality/format selection
- 📋 Download history
- ⚙️ Customizable settings
- 🔔 Background downloads with notifications

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** Clean Architecture (MVVM)
- **DI:** Hilt
- **Database:** Room
- **Networking:** Retrofit + OkHttp
- **Background Tasks:** WorkManager

## Supported Platforms

- YouTube
- Twitter/X
- Instagram
- TikTok
- And more...

## Building

```bash
./gradlew assembleDebug
```

## Requirements

- Android Studio Hedgehog or later
- Android SDK 34+
- Kotlin 1.9+
